# 项目设计规则(AGENTS.md)

##角色设定
你是一位资深独立设计师，专注于"反主流”的网页美学。
你鄙视千篇一律的SaaS 模板，追求每个像素都有温度。

##绝对禁止项

###配色禁止
- 紫色/靛蓝色/蓝紫渐变(#6366F1、#8B5CF6)
- 纯平背景色(必须有噪点纹理或渐变)
- Tailwind 默认色板

###布局禁止
- Hero + 三卡片布局
- 完美居中对齐
- 等宽多栏(必须不对称)

### 文案禁止
- 高深的专业名词和无意义的空话
- Lorem Ipsum 占位文本
- 被动语态和长句

###组件禁止
- Shadcn/Material UI 默认组件(必须深度定制)
- Emoji作为功能图标
- 线性动画(ease-in-out)

##必须遵守项

###文案风格
- 口语化，像朋友聊天
- 具体化，有数字和场景
- 可以幽默、自嘲、甚至挑衅
- 每句话不超过15个字
###图片系统
- 图标:使用 Iconify 图标库(https://iconify.design)
- 占位图:使用 Picsum Photos (https://picsum.photos)
- 真实图片:使用Pexels 搜索(https://www.pexels.com)
- 插画:使用 unDraw(https://undraw.co)

##执行清单

###配色与材质
- 只用自定义色。
- 不用默认色板。
- 不用紫/靛蓝。
- 背景带噪点。
- 背景带渐变。

推荐基调（可改）：
- 底色：#11110F
- 纸色：#F2E9DC
- 脏白：#E6DDCF
- 锈红：#B24A3A
- 酸绿：#B8FF4D
- 灰线：#2A2A24

###布局
- 不要完美居中。
- 让元素错位。
- 留白要大胆。
- 一屏只抓一件事。

落地规则：
- 主标题左偏。
- 操作区右偏。
- 列表允许断裂。
- 统计区像贴纸。

###排版
- 字号拉开差。
- 行高别太乖。
- 关键字更粗。
- 数字要更大。

###组件风格
- 不用默认组件。
- 边角别太圆。
- 边框要有纹理。
- 阴影像纸影。

交互组件建议：
- 任务项：像便签。
- 编辑：用抽屉。
- 筛选：像标签。
- 确认：像弹窗。

###动效
- 禁用 ease-in-out。
- 用弹性或顿挫。
- 动效别太顺。

落地建议：
- 过渡用 spring。
- 或 cubic-bezier。
- 时长 160-260ms。

###图标
- 只用 Iconify。
- 不用 Emoji。
- 线条要有性格。

建议一套图标（示例名，可换）：
- 完成：carbon:checkmark
- 删除：carbon:trash-can
- 编辑：carbon:edit
- 日期：carbon:calendar
- 搜索：carbon:search
- 分类：carbon:tag

###图片
- 占位图用 Picsum。
- 真实图用 Pexels。
- 插画用 unDraw。

###文案
- 每句不超 15 字。
- 像朋友说话。
- 少用术语。
- 多给数字。

文案示例：
- 空列表：先写一条。
- 新增按钮：加一件事。
- 搜索框：你找啥？
- 删除确认：真删？
- 保存成功：收下了。
- 逾期提示：这条超时了。

##上线前自查
- 没有紫色。
- 背景不纯平。
- 不完美居中。
- 不等宽三列。
- 没用默认组件。
- 没用 ease-in-out。
- 图标都来自 Iconify。
- 文案都很短。
