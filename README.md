# Todo - V1.0.0

一个离线可用的待办应用。

项目根目录放的是需求与设计文档。
前端工程在 `web/`。

## 你能做什么

### 任务
- 新增任务。
- 一键完成/取消完成。
- 编辑任务（抽屉）。
- 删除任务（二次确认）。

字段支持：
- 标题（必填）
- 描述
- 截止日期（到日）
- 优先级（P0/P1/P2）
- 分类

### 分类
- 新增分类。
- 重命名分类。
- 删除分类。

删除分类后：
- 该分类下任务会回到“未分类”。

### 找任务
- 状态筛选：全部/未完/已完。
- 优先级筛选：全部/P0/P1/P2。
- 分类筛选：全部/未分类/某分类。
- 搜索：标题+描述匹配。

搜索做了防抖。
不会每敲一个字就卡。

### 统计
- 待办数
- 已完成
- 完成率
- 逾期数（未完成 + 截止早于今天）

### 数据
- 使用 LocalStorage。
- 刷新不丢。
- 写入失败会提示：`存不进去。`

## 快速开始

前置：Node.js + npm。

在 `web/` 里：

- `npm install`
- `npm run dev`

打开：
- `http://localhost:5173/`

## 常用命令

在 `web/` 目录执行：

- `npm run dev`
- `npm run test:run`
- `npm run lint`
- `npm run build`

## 技术栈

- React 19 + TypeScript
- Vite（当前为 rolldown-vite 兼容别名）
- Zustand
- Tailwind CSS（颜色走自定义 token）
- date-fns
- Iconify（图标）
- Vitest + jsdom（测试）

## 目录说明

- `PRD.md`：需求
- `TECH_DESIGN.md`：技术设计
- `AGENTS.md`：视觉与文案规则
- `PAGES_COMPONENTS.md`：页面与组件清单
- `web/`：前端工程
